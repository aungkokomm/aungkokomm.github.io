The 3-stage workflow

CineLibrary Essentials is the first step in a clean, hand-off pipeline that takes a chaotic Downloads folder and turns it into a polished movie library.

1️⃣ Essentials (this app)

Rename messy filenames, organize files into Title (Year)/ folders. Optional casual TMDb scraper for basic metadata.

2️⃣ MediaElch (recommended)

Full-fledged scraping with multiple sources, edition support, trailers, and deeper metadata. Runs against the folders Essentials prepared.

3️⃣ CineLibrary

Scan and browse the finished library across multiple external drives. Reads the .nfo + poster + fanart that MediaElch wrote.

What it does for you

🧠 Smart filename parser

Strips release-group prefixes, codecs (x265, HEVC, AV1), audio (DDP5.1, Atmos), HDR, source tags, and tracker URLs. Extracts the real title and year.

🟥 Word-level diff

Each row shows the original filename with kept tokens grey and removed tokens in red strikethrough — see exactly what was cleaned.

🪄 Inline TMDb search

Per-row 🔍 button opens a TMDb search dialog — pick the right match, app uses that exact ID for metadata download.

📁 Drag-and-drop

Drop a folder onto the window. Recursive scan toggle for whole libraries. Subtitle files (.srt, .sub) follow the rename automatically.

🛠 Bulk power tools

Find & Replace, Smart Title Case, output template picker (Title (Year) or Year - Title), Reset, Select All / None.

⚠️ Safety first

Confirms before destructive ops. Detects duplicates, invalid characters, missing year, TV episodes, and path-too-long issues per row.

Wizard at a glance

  • Step 1 · Clean Names — pick a folder, review/edit cleaned names, click Rename Selected
  • Step 2 · Organize — files auto-carry over, picks Title (Year)/ destination, click Run File to Folder
  • Step 3 · Scrape — auto-list of organized movies, per-row TMDb search, downloads NFO + poster + fanart + actor photos
  • Output is fully Plex / Kodi / Jellyfin / MediaElch compatible
  • Skip Step 3 entirely and use MediaElch — the folders are already in the format it expects
  • TMDb API key embedded — works out of the box, no setup
