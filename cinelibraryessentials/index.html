CineLibrary Essentials

Clean up messy movie and TV-show downloads. Rename, organize, and scrape full metadata into the folder layout Plex, Kodi, Jellyfin and MediaElch expect — fast, offline, no admin required.

The 3-stage workflow

CineLibrary Essentials takes a chaotic Downloads folder and turns it into a polished, library-ready movie collection in a single pass — no other tool required.

1️⃣ Essentials (this app)

Rename, organize and scrape movies and TV shows into the standard Title (Year)/ and Show/Season XX/ layouts — full Kodi NFO with director, writers, country, studios, certification, trailer, complete cast with photos, probed file info, and per-episode metadata for TV.

2️⃣ MediaElch (optional)

If you want to layer in extra sources or edition-specific data, MediaElch reads the folders Essentials prepared. Not required — Essentials now produces equivalent NFO output on its own.

3️⃣ CineLibrary

Scan and browse the finished library across multiple external drives. Reads the full .nfo, poster, fanart, and actor photos Essentials produced.

What it does for you

🧠 Smart filename parser

Strips release-group prefixes, codecs (x265, HEVC, AV1), audio (DDP5.1, Atmos), HDR, source tags, and tracker URLs. Extracts the real title and year.

🟥 Word-level diff

Each row shows the original filename with kept tokens grey and removed tokens in red strikethrough — see exactly what was cleaned.

🪄 Inline TMDb search

Per-row 🔍 button opens a TMDb search dialog — pick the right match, app uses that exact ID for metadata download.

📁 Drag-and-drop

Drop a folder onto the window. Recursive scan toggle for whole libraries. Subtitle files (.srt, .sub) follow the rename automatically.

🛠 Bulk power tools

Find & Replace, Smart Title Case, output template picker (Title (Year) or Year - Title), Reset, Select All / None.

⚠️ Safety first

Confirms before destructive ops. Detects duplicates, invalid characters, missing year, TV episodes, and path-too-long issues per row.

📜 Full Kodi-standard NFO

Director, writers, country, studios, certification, tagline, collection/set, trailer URL, IMDb & TMDb ids, structured ratings, and a probed <fileinfo> block with video codec, duration, and per-track audio & subtitle languages.

🖼 Complete cast with photos

Up to 50 cast members per movie, each with a high-resolution headshot from TMDb saved into .actors/. Downloaded in parallel, idempotent on re-scrape.

🧹 MKV metadata cleanup

Optional in-place scrub of pirate site branding (HDHub4u titles, embedded logos, junk tags) using the bundled mkvpropedit. Only the metadata block is rewritten — the video, audio and subtitles are never re-encoded.

🔔 One-click auto-update

Silent once-per-24h check against GitHub. When a newer release is published, the toast that appears can download and launch the installer for you — no browser, no manual download.

⚙️ Settings page

One place for the output template, recursive scan default, embedded-metadata cleanup default, the TMDb scrape language, and the auto-update preference.

🌐 Scrape in your language

TMDb metadata in Burmese, Hindi, Tamil, Telugu, Thai, Chinese, Japanese, Korean and 8 others — localized titles, plots and posters, with English fallback where needed.

🎬 Edition detection

Director's Cut, Extended, IMAX, 4K Remaster, Theatrical, Unrated, Criterion, Open Matte and more are auto-extracted from messy filenames. Shows as a chip and writes <edition> to the NFO so Plex / Kodi group multiple cuts correctly.

🔎 Rich Movie Details view

Double-tap a scraped movie in Step 3 to open a polished, resizable details window — hero fanart, floating poster, full plot, color-coded director / writer / studio / country / genres / IDs / file info, scrollable cast strip with photos, plus one-click Play and Open Folder.

📺 TV show support

Full pipeline for TV: cleans episodes to Show - S01E01 - Title, organizes into Show/Season XX/, and scrapes tvshow.nfo + show artwork + per-episode NFO and thumbnails. Recognizes flat "Season Complete" folders too.

✅ Verify your library

Fill gaps only sweeps an existing library and scrapes just the folders missing an NFO, poster, fanart, or actor photos — skipping the ones already complete.

Wizard at a glance

  • Mode selector — Auto · Movies · TV Shows, so the app always knows what you're processing
  • Step 1 · Clean Names — review/edit cleaned names (movies → Title (Year), TV → Show - S01E01 - Title), click Rename Selected. Optional MKV metadata scrub kills embedded pirate branding in place.
  • Step 2 · Organize — files auto-carry over into Title (Year)/ or Show/Season XX/, with folder merging and subtitle handling, then Run File to Folder
  • Step 3 · Scrape — full Kodi-standard NFO + original-resolution poster, fanart, and complete cast photos for movies; tvshow.nfo + per-episode NFO and thumbnails for TV. Fill gaps only verifies an existing library.
  • Output is fully Plex / Kodi / Jellyfin / MediaElch / CineLibrary compatible
  • TMDb API key embedded — works out of the box, no setup · 16 scrape languages
