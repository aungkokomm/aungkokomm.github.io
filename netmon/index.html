NetMon

A lightweight network bandwidth monitor that lives in the corner of your screen — DU Meter style, but free, modern, and built for Windows 11.

⬇ Download v1.4 View source
C# / WinForms · .NET 8 · 13 ★ on GitHub · MIT licensed

Why NetMon

🪶 Tiny footprint

Single-window widget, draggable, always-on-top optional. Sub-megabyte RAM, near-zero CPU.

📈 Real-time graph

Live up/down speed plot. Adjustable refresh rate. Reads directly from Windows performance counters.

🔌 VPN-aware

v1.4 fixes speed reporting under VPN tunnels — counts the actual interface, not the underlying NIC.

⚙️ No bloat

No telemetry, no ads, no "pro" tier. Just a number and a graph.
