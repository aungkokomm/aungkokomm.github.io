CineLibrary

Native Windows catalog for your MediaElch-scraped movies and TV shows. Browse, search, filter, and play across any number of external drives — fast, offline, portable.

What's new in v2.8.1 — TV Shows 📺

  • Lists hold movies and shows together. My Lists are independent buckets — add a show with 📑 Add to list; opening a list shows its TV shows in a row above the movie grid.
  • Per-episode details. Single-click an episode for full plot, air date, rating, runtime, and tech info (resolution, HDR, codecs, audio languages, subtitles, container, file size).
  • TV in Statistics. A new TV Shows panel shows total shows, episodes, combined runtime, average rating, and episodes-watched progress.
  • Richer season headers — Play season + Mark all watched/unwatched, with episode/watched/runtime counts.
  • TV shows, alongside your movies. Any folder with a tvshow.nfo is detected as a show; episodes are read from the SxxExx in each filename. A single drive can hold both — movies are unaffected.
  • One show page, no drilling. Poster, plot, year · rating · status, a watched roll-up, genre chips and a cast strip up top; below it, each season is a horizontal row of episode cards (Netflix-style).
  • ▶ Play next unwatched — jumps straight to the first unwatched episode; reads "✓ All watched" when you're done.
  • Per-episode watched, show-level favorite / watchlist. Toggle watched right on each episode card; the progress roll-up updates instantly. Double-click or ▶ Play opens it in your default player (and marks it watched).
  • Travels with the drive, just like movies — a per-show cinelibrary-state.json mirrors favorite / watchlist / note + per-episode watched, so removing and re-adding a drive brings your progress back.
  • Drives page shows a purple SHOWS count pill beside the gold MOVIES pill.
  • Browse back button — drilling into By Genre / Decade / Rating / Studio / Collections now gives you a "‹ Back" to that page.
  • Everything from earlier versions (drive-portable personal state, multi-select, brand polish, motion, active filter chips, light-theme tightening) carries forward unchanged.

Why CineLibrary

🚀 Native, fast

WinUI 3 + C#. Cold start under a second. Scrolling through 1,000+ posters stays smooth thanks to an LRU image cache.

💾 Multi-drive aware

Index a USB drive, unplug it — its movies fade to "offline" instead of vanishing. Plug it back in and they wake up via WM_DEVICECHANGE.

📦 MediaElch-friendly

Reads .nfo + poster/fanart files alongside your videos. No re-scraping, no duplicate metadata.

🔌 Portable, offline

Install anywhere — external drive, USB stick, Documents. Library data lives next to the exe. No telemetry.

🎨 Light + dark

Theme-aware design, follows system theme by default. Mica backdrop on Windows 11.

🔄 Auto update notifier

Checks GitHub for new releases on startup. Shows a quiet toast — never installs silently, never nags.

Features

  • Library, Favorites, Watchlist, Statistics
  • Filter by collection, genre, decade, watched status
  • Search title, actor, director, plot
  • Grid + list views with S/M/L/XL density
  • Movie details: cast, crew, ratings, runtime, file path
  • Toggle watched / favorite / watchlist with one click
  • Export catalog as CSV or HTML
  • Keyboard shortcuts: Ctrl+F search, Ctrl+B sidebar, Ctrl+Q quit, Ctrl+Shift+/ help
