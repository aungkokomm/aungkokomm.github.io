CineLibrary

What's new in v2.7.0 — Personal state travels with your drives

  • Watched, favorites, watchlist, lists and notes now ride along with your drive. Every movie folder gets a tiny cinelibrary-state.json next to the .nfo capturing its personal state.
  • Remove a drive → re-add it later → everything's back. On the next scan, CineLibrary reads each movie's sidecar and merges the values back in.
  • Safer Remove Drive dialog — shows how many movies on the drive carry personal data and offers a checkbox (default ON) to save that state to the drive first. If the drive is offline, a clear warning recommends cancelling until you reconnect.
  • New per-drive "Sync state" button — push everything to disk on demand (handy before handing the drive to a friend or after restoring a backup).
  • Live mirroring — every Mark Watched / Favorite / Watchlist / Play / Add to list / Note save writes the sidecar immediately. A quiet startup sweep catches anything edited while a drive was offline.
  • Conflict rule: DB wins where it already has a value; lists are additive — joined when missing, never silently removed.
  • Your .nfo files are never touched. The only file CineLibrary writes inside a movie folder is cinelibrary-state.json. Posters, fanart, videos — all untouched.
  • Everything from v2.6 (brand wordmark, modernised drives page, motion, active filter chips, multi-select, light-theme polish) carries forward unchanged.

Why CineLibrary

🚀 Native, fast

WinUI 3 + C#. Cold start under a second. Scrolling through 1,000+ posters stays smooth thanks to an LRU image cache.

💾 Multi-drive aware

Index a USB drive, unplug it — its movies fade to "offline" instead of vanishing. Plug it back in and they wake up via WM_DEVICECHANGE.

📦 MediaElch-friendly

Reads .nfo + poster/fanart files alongside your videos. No re-scraping, no duplicate metadata.

🔌 Portable, offline

Install anywhere — external drive, USB stick, Documents. Library data lives next to the exe. No telemetry.

🎨 Light + dark

Theme-aware design, follows system theme by default. Mica backdrop on Windows 11.

🔄 Auto update notifier

Checks GitHub for new releases on startup. Shows a quiet toast — never installs silently, never nags.

Features

  • Library, Favorites, Watchlist, Statistics
  • Filter by collection, genre, decade, watched status
  • Search title, actor, director, plot
  • Grid + list views with S/M/L/XL density
  • Movie details: cast, crew, ratings, runtime, file path
  • Toggle watched / favorite / watchlist with one click
  • Export catalog as CSV or HTML
  • Keyboard shortcuts: Ctrl+F search, Ctrl+B sidebar, Ctrl+Q quit, Ctrl+Shift+/ help
