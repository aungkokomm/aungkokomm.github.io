What's new in v2.9 — Discovery, Memory, Polish ✨

A big drop spanning three themes — plus a small v2.9.1 follow-up that makes Recently Added actually mean recent.

🎬 Discovery — new ways to find what to watch

  • 🟣 Continue Watching for TV — a row at the top of the TV Shows page showing the next unwatched episode for every show you've started. One click jumps straight back in.
  • 🕓 Recently Watched — new sidebar entry that lists everything you've played, freshest first.
  • 📅 On This Day — surfaces movies you watched on this calendar date in past years, plus movies released on this date in past years. Even on a fresh install you'll get cinema anniversaries most days.
  • 🎲 Filter-aware Surprise Me — new dice button on the Library toolbar picks a random movie from the current filtered view.
  • 🔍 TV in global search — the search box now matches TV shows alongside movies.
  • 🆕 Recently Added, capped at 50 (v2.9.1) — the section now stops at the 50 most recently added titles, so the count is honest and the view actually means what it says. A removable filter chip lets you drop the cap.

🧠 Memory — your watching, remembered

  • Watch history with dates — every play / mark-watched is logged with a timestamp. A new "Last Watched ↓" sort and the Recently Watched + On This Day surfaces above are built on top.
  • 🏷 Free-form tags — tag movies and shows with anything ("rewatched", "comfort", "Christmas", "long-haul flight"). Tags appear in the sidebar with counts and filter the library with one click. Travels with the drive like favorites and notes.
  • ★ Per-episode favorite & note — mark individual episodes as favorites and jot a note. Episode cards show a ★ badge when favorited.

✨ Polish — the little but lovely things

  • ⌨️ More keyboard shortcuts — / focuses search, F toggles favorite, W toggles watchlist, Delete removes from current list.
  • 💾 Backup / restore — one-click JSON export of everything personal: favorites, watchlist, notes, lists, tags, watched flags, last-played times, per-episode state, and watch history. Importable on another PC with additive merge — re-importing is a safe no-op.
  • Everything from v2.8.x (TV-show support, per-episode details, multi-drive portable state, brand polish) carries forward unchanged.

Why CineLibrary

🚀 Native, fast

WinUI 3 + C#. Cold start under a second. Scrolling through 1,000+ posters stays smooth thanks to an LRU image cache.

💾 Multi-drive aware

Index a USB drive, unplug it — its movies fade to "offline" instead of vanishing. Plug it back in and they wake up via WM_DEVICECHANGE.

📦 MediaElch-friendly

Reads .nfo + poster/fanart files alongside your videos. No re-scraping, no duplicate metadata.

🔌 Portable, offline

Install anywhere — external drive, USB stick, Documents. Library data lives next to the exe. No telemetry.

🎨 Light + dark

Theme-aware design, follows system theme by default. Mica backdrop on Windows 11.

🔄 Auto update notifier

Checks GitHub for new releases on startup. Shows a quiet toast — never installs silently, never nags.

Features

  • Library, Favorites, Watchlist, Statistics
  • Filter by collection, genre, decade, watched status
  • Search title, actor, director, plot
  • Grid + list views with S/M/L/XL density
  • Movie details: cast, crew, ratings, runtime, file path
  • Toggle watched / favorite / watchlist with one click
  • Export catalog as CSV or HTML
  • Keyboard shortcuts: Ctrl+F search, Ctrl+B sidebar, Ctrl+Q quit, Ctrl+Shift+/ help
